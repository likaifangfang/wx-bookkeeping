# PRD - 微信记账小程序

> 版本：v1.0
> 日期：2026-06-16

## 1. 产品定位

一款轻量、好用的微信记账小程序，主打 **个人 + 共享账本** 两种使用场景，零成本运维、免费使用。

**一句话描述**：在微信里随手记一笔，自己看得清，伙伴一起记。

## 2. 目标用户

| 用户画像 | 典型场景 |
|---------|---------|
| 个人用户 | 想记录日常开销，看月度收支趋势 |
| 情侣 / 夫妻 | 共同管理生活开销，AA 或共同账本 |
| 家庭 | 家庭成员共同记录家庭收支 |
| 朋友合租 | 合租生活费用分摊 |

## 3. 核心价值

- **快**：3 秒记一笔，比同类应用更轻
- **共享**：原生支持多人协作账本
- **免费**：无广告、无会员、不卖数据
- **安全**：数据加密、用户级隔离

## 4. 用户角色与权限

| 角色 | 权限 |
|------|------|
| 普通用户 | 注册即拥有一个"个人账本" |
| 账本所有者（Owner） | 创建账本、邀请成员、移除成员、删除账本、所有记录的增删改查 |
| 账本编辑者（Editor） | 在账本中记账、修改自己的记录、查看账本所有数据 |
| 账本查看者（Viewer） | 只能查看账本数据，不能记账 |

> 第一版先实现 Owner / Editor 两种角色，Viewer 后续迭代。

## 6. 功能详细描述

### 6.1 记一笔

**优先级**：P0（核心）

**功能要点**：
- 切换"支出 / 收入"
- 输入金额（数字键盘，自带小数点）
- 选择分类（横向滑动图标列表，常用置顶）
- 选择日期（默认今天，可改）
- 选择账本（默认当前账本，可切换）
- 备注（可选，最多 100 字）
- 上传图片（可选，最多 3 张，记小票）

**交互要点**：
- 入口必须**1 步可达**：底 tab 中间有醒目"+"按钮
- 记录成功后停留在记账页，方便连续记账
- 顶部小提示："已记录，可继续"

### 6.2 账单列表与详情

**优先级**：P0

**列表页**：
- 按日期倒序分组
- 每天显示当天小计（收入 / 支出）
- 顶部显示当月小计
- 支持下拉刷新、上拉加载更多
- 支持账本切换（顶部账本选择器）
- 支持搜索、筛选（按分类、按时间段、按金额范围）

**详情页**：
- 显示完整字段
- 编辑、删除（编辑会留修改记录，删除二次确认）

### 6.3 账本管理

**优先级**：P0

**列表**：
- 我创建的账本
- 我加入的账本
- 显示账本名、成员数、本月收支

**新建账本**：
- 名称、图标、颜色、币种（默认人民币）
- 类型：个人 / 共享

**共享账本**：
- 邀请成员：分享小程序卡片，对方点击进入即加入
- 成员管理：所有者可移除成员、转让所有权
- 退出账本：成员可主动退出

### 6.4 分类管理

**优先级**：P1

**默认分类**（账本初始化时自动创建）：

支出类：
- 餐饮、交通、购物、娱乐、居家、医疗、教育、通讯、人情、其他

收入类：
- 工资、奖金、理财、退款、红包、其他

**自定义**：
- 新增分类（名称、图标、颜色）
- 编辑、排序、隐藏分类（不删，避免历史数据丢失）

### 6.5 统计分析

**优先级**：P1

**首页统计卡片**：
- 本月支出、本月收入、本月结余

**统计页**：
- 时间维度：日 / 周 / 月 / 年 / 自定义
- 视图：
  - 收支趋势图（折线 / 柱状）
  - 分类占比图（饼图）
  - 分类排行榜（Top 10）
- 支持账本切换

### 6.6 我的

**优先级**：P1

- 头像、昵称（取微信信息）
- 我的账本
- 设置（默认账本、隐私模式、提醒时间）
- 关于、反馈、版本号

## 7. 用户主流程

### 7.1 新用户首次使用
```
进入小程序
  ↓
微信授权登录（自动）
  ↓
自动创建"我的账本"
  ↓
进入首页（空状态引导）
  ↓
点击"+"记一笔
```

### 7.2 邀请共享账本
```
账本主点击"邀请"
  ↓
分享小程序卡片到微信好友/群
  ↓
对方点击卡片进入小程序
  ↓
弹出"是否加入 XX 账本"
  ↓
确认 → 加入成功 → 进入该账本
```

## 8. 非功能需求

- **性能**：首页打开 2 秒内、记账提交 1 秒内
- **离线**：弱网环境下记账先存本地，恢复后自动同步（v2 再做）
- **隐私**：用户数据仅自己和授权成员可见，不做任何商业分析
- **兼容性**：微信基础库 2.20+，覆盖主流机型

## 9. 不做的事（边界）

明确**第一版不做**的功能，避免范围蔓延：

- ❌ 多币种 / 汇率
- ❌ 资产 / 负债 / 投资管理
- ❌ AA 自动分账计算
- ❌ 银行卡导入、自动同步
- ❌ OCR 识别小票
- ❌ 预算管理（v2 再做）
- ❌ 数据导出 Excel（v2 再做）
- ❌ 周期性账单（订阅、房租等，v2 再做）
